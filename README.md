植物监测与控制系统 用户手册
1. 产品简介
本系统是一套用于植物环境监测与远程控制的智能设备，支持：
• 实时采集温度、空气湿度、土壤湿度、光照强度
• 本地 LCD 显示运行状态与环境数据
• 通过 OneNET 云平台上传数据
• 通过网页远程控制风扇、水泵、补光灯
• 支持自动模式与远程模式切换
适用场景：
• 室内盆栽养护
• 小型温室环境监测
• 教学演示与课程设计
• 智能农业入门实验
2. 系统组成
本系统主要由以下部分组成：
• 主控板：STM32F103
• 通信模块：ESP8266
• 传感器：
• DHT11 温湿度传感器
• 土壤湿度传感器
• 光敏传感器
• 执行器：
• 风扇
• 水泵
• 补光灯
• 显示模块：LCD 屏幕
• 云平台：OneNET
• 控制终端：网页控制面板
3. 主要功能
系统具备以下功能：
• 采集环境数据并本地显示
• 周期上传数据到云平台
• 通过网页查看实时状态
• 远程控制风扇、水泵、补光灯
• 自动模式下根据阈值自主控制设备
• 可从远程模式恢复到自动模式
4. 上电使用流程
1. 给设备上电。
2. 观察 LCD 和串口输出，等待系统完成初始化。
3. 初始化成功后，设备自动连接 WiFi 和 OneNET 平台。
4. 正常运行后，LCD 会显示环境数据，网页也可查看实时状态。
串口正常启动时会显示类似内容：
• Init ESP8266...
• Init MQTT...
• MQTT connecting...
• Subscribe property set topic...
如果能看到这些内容并进入数据打印阶段，说明设备连接成功。
5. LCD 显示说明
LCD 用于显示设备运行状态和传感器数据，主要包含：
• 温度
• 空气湿度
• 土壤湿度
• 光照强度
• 风扇状态
• 水泵状态
• 补光灯状态
LCD 还支持多页面切换，可查看：
• 环境数据页
• 本地控制页
• 阈值设置页
6. 本地按键说明
设备带有本地按键，可用于页面切换和本地操作。
按键功能：
• S1
切换 LCD 页面
• S2
切换控制模式或选择设置项
• S3
增加参数 / 切换项目
• S4
减少参数 / 切换项目
通过本地按键可实现：
• 翻页查看状态
• 本地手动开关风扇、水泵、补光灯
• 修改自动控制阈值
7. 自动模式说明
自动模式下，系统会根据当前环境数据自动控制设备：
• 温度高于阈值时，风扇自动开启
• 土壤湿度低于阈值时，水泵自动开启
• 光照强度低于阈值时，补光灯自动开启
适合日常无人值守运行。
8. 远程模式说明
当用户通过网页或云平台远程下发控制指令后，系统会进入远程模式。
远程模式下：
• 风扇、水泵、补光灯按远程命令执行
• 自动控制逻辑暂停
支持远程控制的项目：
• 风扇
• 水泵
• 补光灯
适合临时人工干预。
9. 恢复自动模式
当系统处于远程模式时，可以通过网页上的：
• 恢复自动
按钮，将设备切换回自动模式。
恢复自动模式后：
• 设备退出远程控制
• 自动控制逻辑重新生效
如果网页点击后未恢复，请检查：
• 设备是否在线
• OneNET 是否连接正常
• 云端 auto_mode 属性是否已配置
10. 网页端使用方法
网页端可用于远程查看数据和控制设备。
网页支持：
• 查看实时环境数据
• 远程控制风扇、水泵、补光灯
• 点击“恢复自动”回到自动模式
• 手动刷新数据
• 修改连接参数
使用步骤：
1. 打开网页控制面板。
2. 等待设备数据加载完成。
3. 查看当前温度、湿度、光照等数据。
4. 如需远程控制，点击对应开关。
5. 控制结束后，点击“恢复自动”。
11. OneNET 平台说明
系统通过 OneNET 进行数据上传与指令下发。
平台中常见属性包括：
• temperature 温度
• air_humidity 空气湿度
• soil_humidity 土壤湿度
• lighting_intensity 光照强度
• fan 风扇状态
• bump 水泵状态
• LED 补光灯状态
• auto_mode 自动模式状态
说明：
• auto_mode = true 表示自动模式
• auto_mode = false 表示远程模式
12. 正常运行现象
系统正常运行时应满足：
• LCD 正常显示数据
• 串口持续打印环境信息
• OneNET 平台数据持续更新
• 网页能看到实时数据
• 远程控制命令能够生效
• 点击“恢复自动”可回到自动控制状态
13. 常见问题与处理
• 上电后 LCD 白屏
检查供电、电路连接和屏幕初始化是否正常。
• 串口乱码
检查波特率设置是否一致，当前系统串口为 115200。
• 网页没有数据
检查：
• 设备是否联网
• OneNET 是否连接成功
• product_id、device_name、access_key 配置是否正确
• 云平台有数据但网页无数据
检查网页中的接口配置和浏览器网络访问情况。
• 远程控制无效
检查：
• 设备是否在线
• 云端属性是否配置正确
• ESP8266 是否正常连接云平台
• 恢复自动模式失败
检查：
• auto_mode 是否已在 OneNET 物模型中配置为布尔型可读写属性
• 设备是否在线
• 当前是否能正常接收远程控制命令
14. 使用建议
建议按以下方式使用：
1. 日常保持自动模式运行。
2. 仅在需要人工干预时使用远程控制。
3. 远程控制结束后，及时点击“恢复自动”。
4. 定期检查传感器状态和执行器是否正常工作。
5. 如长期运行，建议保持稳定供电和网络环境。
15. 安全注意事项
• 水泵使用时注意防止空转
• 补光灯和风扇需确保供电稳定
• 避免潮湿环境中主控板裸露
• 不要带电插拔关键模块
• 修改参数前建议先确认设备在线和状态正常
